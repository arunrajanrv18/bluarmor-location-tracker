# BluArmor Location Tracker — Config Comparison

Interactive visualization comparing GPS location tracker performance
across 6 Android devices and 5 update intervals (1s, 5s, 15s, 30s, 1min).

## Metrics covered
- GPS Accuracy (p50 and p95)
- Battery drain (%/h)
- Max gap between samples (ms)
- Mean device temperature (°C)
